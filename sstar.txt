TSLIB编译：
1、从 https://github.com/libts/tslib/releases/tag/1.15 中下载 tslib-1.15tar.bz2 到Linux系统下并解压，进入解压目录，新建install目录
2、./configure --prefix=/home/ronnie/work/ssd201/qt/tslib-1.15/install --host=arm-linux-gnueabihf     （/home/ronnie/work/ssd201/qt/tslib-1.15/install为install的路径，按本地的来）
3、make
4、make install
QT编译
1、从 http://download.qt.io/archive/qt/5.15/5.15.0/single/ 下载 qt-everywhere-src-5.15.0.tar.xz 到Linux系统下并解压   tar xf
2、设置make.sh内--prefix=参数，指定install目录 提前设置好，构建install目录
3、修改qt-everywhere-src-5.15.0/qtbase/mkspecs/linux-arm-gnueabi-g++/qmake.conf qmake.conf文件
4、执行./make.sh
5、执行make   （注意缺少g++  先执行 apt-get install g++）
6、执行make install
vi ~/.bashrc
export PATH=/home/run/qt/install/bin:$PATH
source ~/.bashrc
